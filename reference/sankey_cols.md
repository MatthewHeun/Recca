# Sankey diagram data frame column names

A string list containing named names of columns in Sankey data frames.
Items in the list provide default values for column name function
arguments throughout the `Recca` package.

## Usage

``` r
sankey_cols
```

## Format

A string list with 1 entries.

- sankey:

  The name of a column in a wide-by-matrices data frame containing
  Sankey diagrams.

## Examples

``` r
sankey_cols
#> $sankey
#> [1] "Sankey"
#> 
```

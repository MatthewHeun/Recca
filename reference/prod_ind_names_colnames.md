# Names of data frame columns containing Product and Industry matrix row and column names

A list of data frame column names. The data frame columns contain
vectors of unique Product and Industry column names from **R**, **U**,
**V**, and **Y** matrices.

## Usage

``` r
prod_ind_names_colnames
```

## Format

A list with 2 entries.

- product_names:

  The name of a data frame column that contains Products. Default is
  "Product.names".

- industry_names:

  The name of a data frame column that contains Industries. Default is
  "Industry.names".

# Names of row and column types in PSUT matrices

A list of row and column types.

## Usage

``` r
row_col_types
```

## Format

A list with 2 entries.

- product_type:

  The name of the product row and column type. Default is "Product".

- industry_type:

  The name of the industry row and column type. Default is "Industry".

- unit_type:

  The name of the unit row and column type. Default is "Unit".

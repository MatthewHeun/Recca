# A list of High pressure matter (HPM) useful work products

A list of High pressure matter (HPM) useful work products.

## Usage

``` r
hpm_eu.products
```

## Format

A list with 3 entries.

- HPA:

  The useful work product (Eu.product) "High pressure air"

- HPL:

  The useful work product (Eu.product) "High pressure liquids"

- HPNG:

  The useful work product (Eu.product) "High pressure natural gas"

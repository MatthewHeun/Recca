# A list of Mechanical useful work products

A list of Mechanical useful work products

## Usage

``` r
mechanical_eu.products
```

## Format

A list with 4 entries.

- MD:

  The useful work product (Eu.product) Mechanical drive

- KE:

  The useful work product (Eu.product) Kinetic energy

- MW:

  The useful work product (Eu.product) Mechanical work

- MF:

  The useful work product (Eu.product) Material fracture

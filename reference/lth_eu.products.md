# A list of Low temperature heat (LTH) useful work products

A list of Low temperature heat (LTH) useful work products

## Usage

``` r
lth_eu.products
```

## Format

A list with 2 entries.

- LTH.20.C:

  The useful work product (Eu.product) "LTH.20.C"

- LTH.60.C:

  The useful work product (Eu.product) "LTH.60.C"

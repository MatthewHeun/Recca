# A list of Propulsion useful work products

A list of Propulsion useful work products

## Usage

``` r
propulsion_eu.products
```

## Format

A list with 3 entries.

- MP:

  The useful work product (Eu.product) Marine propulsion

- RoP:

  The useful work product (Eu.product) Road propulsion

- RaP:

  The useful work product (Eu.product) Rail propulsion

# A list of Medium temperature heat (MTH) useful work products

A list of Medium temperature heat (MTH) useful work products

## Usage

``` r
mth_eu.products
```

## Format

A list with 2 entries.

- MTH.100.C:

  The useful work product (Eu.product) "MTH.100.C"

- MTH.200.C:

  The useful work product (Eu.product) "MTH.200.C"
